2. Changes to the specification

Navbar transparency: On the home page, the navbar was made transparent in order to blend better with the background. This decision was made for aesthetic reasons. This was implemented as a CSS rule specifically for the home.html document.

Font sizes: Font sizes in general were changed due to the mockups being made in Photoshop, which did not measure the size of elements in screen-size pixels. Sizes in pixels were changed into relative "em" sizes, making it more predictable when working with high DPI screens such as mobile devices.

Mobile fixes: In order to further facilitate a mobile version, a minimized version of the navbar (commonly referred to as a hamburger) was added when browsing from a mobile device. The usability of this version was significantly better than the normal navbar, because it does not require precision to hit the links. In addition to this, margins and other spacing was reduced when browsing from a high-resolution device in order to make it more readable.

Removing a header image: On the video.html page, the header image was removed, seeing as it would be superfluous when there are videos on the page.

Adding shadows: On the home.html page, shadow was added to the large Excellence text in the center of the page, in order to make it more visible.
